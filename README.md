  Antes de empezar, asegúrate de tener en tu dispositivo:

Java 17 o superior.

Maven (generalmente viene incluido en IntelliJ/Eclipse).

Docker Desktop (Recomendado para la base de datos).

Navegador web (Chrome/Edge recomendado).


En la terminal de docker desktop, dentro de app-gestion-de-gastos, ejecuta el siguiente comando:

docker compose up -d

Abre el proyecto en tu IDE

Localiza la clase GestorApp.java en src/main/java/com/appgestor/.

Haz clic derecho y selecciona Run.

Y...¡listo! el servidor estará corriendo, solo abre index.html con live server y podrás utilizarlo 


Aqui tienes los ejemplos de usuario para el admin y superadmin, pero puedes registrar tu usuario sin problema:
